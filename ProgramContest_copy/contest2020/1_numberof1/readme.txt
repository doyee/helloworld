#题目: 输入一个整数N，
（1）输出该数的十六进制表示
（2）输出该数二进制表示中1的个数
备注：如果N是负数，则以补码的形式计算。

#限制：
输入的N不能为字符

#例子1
Input
35

Output
0x23
3

#例子2
Input
a

Output
input is not valid

a. compiling command: gcc numberof1.c
