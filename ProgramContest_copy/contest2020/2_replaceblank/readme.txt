#题目: 输入一个字符串,要求：
（1）输出字符串中空格的数量
（2）替换字符串内的空格为下划线"_"，输出调整后的字符串
     如果没有空格键，则输出原来的字符串
（3）输出字符串的长度小于等于20，否则返回错误

#限制：
输入的字符串长度要小于等于20

#例子1
Input
asdfsadfasd22gad3asdfsad4 sadfasdfasdfsadf 5672a

Output
blank number =2
string length>MAX lenght(20)

#例子2
Input
asdf  asdf

Output
blank number =2
asdf__asdf
