#Title: string basic operations

Given a string(S):
1. delete all duplicated characters
2. delete all non-letter & non-digit characters
3. covert all lowercase letters to capital, and covert all capital letters to lowercase
4. sort all characters in descending order by ASCII code

#Standard Input
The input string(S)

#Constraints:
0 < N < 100

#Standard Output
The result string

#Sample
Input
abbcd123JKL%

Output
lkjDCBA321
