#Title:FLOAT sum operations

Given a senquence(N):
1. observe the mathematical rule
2. calculate the sum of the sequence N
   
  2/1, 3/2, 5/3, 8/5, 13/8, 21/13,...


#Standard Input
The input integer N

#Constraints: //输入限制检查1分
0 < N < 30

#Standard Output
The sum of 2/1, 3/2, 5/3, 8/5, 13/8, 21/13,....

#Sample
Input
3

Output        //输出表达式正确2分,结果正确7分
2/1+3/2+5/3 = 5.166667
